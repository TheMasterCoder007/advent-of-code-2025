# Advent of Code - 2025

This repository contains my solutions for the 2025 Advent of Code challenges. All challenges will be completed in TypeScript and are done purely for fun and skill development.

## Requirements
- Node.js (recommend v22+)
- Webstorm, VS Code, or preferred code editor

## Setup
- clone project to your local device
- if corepack is not already installed run `npm install -g corepack`
- run `corepack enable`
- run `yarn install`

## Run
- run yarn + day# (example: `yarn day1`)
